*inlineblock
Baseline alignment remains stable unless text alignment/wrapping changes due to padding.
If one element’s text wraps to 2 lines, then its baseline is at the bottom of the second line → so that element appears lower.
If padding makes the content area look taller/shorter, it creates the illusion that one box is “jumping up or down,” but really the baseline is untouched.
