| Method                           | Example                                   | Returns        | Array?         | Notes           |
| -------------------------------- | ----------------------------------------- | -------------- | -------------- | --------------- |
| `getElementsByTagName("li")`     | `document.getElementsByTagName("li")`     | HTMLCollection | ❌ (array-like) | Live collection |
| `getElementsByClassName("name")` | `document.getElementsByClassName("name")` | HTMLCollection | ❌ (array-like) | Live collection |
| `querySelectorAll("li")`         | `document.querySelectorAll("li")`         | NodeList       | ❌ (array-like) | Static snapshot |


| Feature             | HTMLCollection                                   | NodeList                                   |
| ------------------- | ------------------------------------------------ | ------------------------------------------ |
| Returned by         | `getElementsByTagName`, `getElementsByClassName` | `querySelectorAll`, `childNodes`           |
| Type                | HTMLCollection                                   | NodeList                                   |
| Live?               | ✅ Yes                                            | ⚠️ Usually No                              |
| Contains            | Only HTML elements                               | Can contain any node (text, comment, etc.) |
| Index access        | ✅ Yes (`[0]`)                                    | ✅ Yes (`[0]`)                              |
| `.forEach()` method | ❌ No                                             | ✅ Yes (in modern browsers)                 |
| Convert to Array    | `[...collection]` or `Array.from(collection)`    | Same                                       |
