How to get current date : time is taken from our device.
method 1 :
* datatype now = new Date();
console.log(now);  // UTC time!
console.log(now.toISOString()); // UTC time
console.log(now.toString()); // Our device time in string style! 24hr
console.log(now.toLocaleString()); // India/location formatted style. 12hr clock

// According to Local Time
console.log(now.getDate());
console.log(now.getDay()); // 1 to 7
console.log(now.getTime());
console.log(now.getMonth());// 0 to 11
....

// month 0 to 11, days 1 to 7/
