** .filter() is an array method in JavaScript that:
Tests each element of an array using a callback function (condition).
Creates a new array containing only the elements that satisfy (return true for) the condition.
Does not modify the original array.

* USED to filter array by function-condition! 
    if condition true then it select & add in new return array.
    if condition false not included!

const arrayName = [90,2,1,4334,54,664,32];

* .filter() returns a new array ! we can store it in newvariable or print it directly.
let newvariable = arrayName.filter((number)=>number>25);

* .some(()=> === ); // atleast one element in the array passes the test return true else false.
    const arr = [1, 3, 5];
    console.log(arr.some(num => num > 4)); // true (because 5 > 4)
* .every(()=> ===); // every elements should pass the test then true else false.
    const arr = [2, 4, 6];
    console.log(arr.every(num => num % 2 === 0)); // true
